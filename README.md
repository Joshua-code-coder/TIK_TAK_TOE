# TIK_TAK_TOE
Tic Tac Toe (Python Tkinter)

A graphical Tic Tac Toe game built using Python and Tkinter.
The game supports both 1v1 multiplayer mode and Player vs Bot mode with an AI opponent using the Minimax algorithm.

This project demonstrates GUI development, game logic implementation, and basic AI in Python.

Features

🎮 Two Game Modes

Player vs Player (1v1)

Player vs Bot (AI opponent)

🤖 AI Bot

Uses the Minimax algorithm to choose the best move.

🏆 Best of 3 Rounds

Game continues for up to 3 rounds.

Player who wins 2 rounds first wins the game.

🎨 Customization

Change the button background color.

🟨 Winning Highlight

Winning combination is highlighted on the board.

🔁 Game Controls

Restart round

Restart entire game

Exit game

🧠 Draw Detection

Detects draw situations and proceeds to next round.

Technologies Used

Python

Tkinter (GUI Library)

Minimax Algorithm (AI Logic)

How the Game Works

Enter usernames for both players.

Choose the game mode:

Bot Mode → Play against AI.

1v1 Mode → Two players play on the same computer.

Players take turns placing X and O on the board.

A round ends when:

A player gets 3 in a row, or

The board becomes full (draw).

The game continues for up to 3 rounds.

The player with the most wins becomes the overall winner.
